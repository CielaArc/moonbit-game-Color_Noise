# ColorNoise：色噪v3.0
## 游戏简介
这是一次奇异的经历，你可以认为是一场梦、或是一次奇遇。
你来到了一个光怪陆离的世界，它失去色彩并且充满噪音，你感受到自己日渐疯狂、难以忍受，但还好，你还有手中的武器。
去消灭那个疯狂的星球状怪物：科洛诺斯
将世界复原，找寻回家的路吧。
